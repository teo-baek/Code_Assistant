# Code-Assistant: 로컬 기반 AI 코딩 어시스턴트

Code-Assistant는 단순한 코드 검색 도구가 아닙니다. 당신의 프로젝트 전체 맥락을 이해하고, 스스로 생각하며, 사용할수록 똑똑해지는 로컬 기반의 AI 협업 파트너입니다.

보안이 중요한 기업 환경이나 폐쇄망(Air-gapped)에서도 안전하게 사용할 수 있도록 모든 데이터와 추론 과정이 100% 로컬에서 이루어집니다.

## 핵심 비전 (Vision)

"도메인 장벽을 허무는 협업(Collaboration across Domains)"

백엔드 개발자가 프론트엔드 코드를 수정해야 할 때, 혹은 앱 개발자가 DB 스키마를 이해해야 할 때 겪는 어려움을 해결합니다. 이 도구는 Tech Lead처럼 프로젝트 전체 구조를 파악하고, 각 파일이 어떻게 연결되는지 상세하게 설명하며 가이드합니다.
